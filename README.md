# login-page

O presente repositório tem o código de uma página de login em HTML com estilos CSS. A página é um clone de um design requisitado em um desafio de código e demonstra uma interface moderna e responsiva.

## Funcionalidades

- Simular o login com uso de e-mail e senha.
- Campos de entrada otimizados:
  a) Campo e-mail: O campo de e-mail é do tipo email, permitindo a entrada de endereço de email e validação simples do formato email.
  b) Campo senha: O campo de senha é do tipo password, ocultando os caracteres digitados para segurança.

- Ocultamento de senha durante a digitação com caracteres, garantindo privacidade.

- Estilização com CSS : Utiliza um arquivo CSS externo (style.css) para definir o layout, cores, fontes e efeitos visuais.

- Design : O design é responsivo e centrado para uma melhor experiência do usuário.

## Fonte Roboto

A fonte Roboto é usada para atender ao design apresentado e representa uma fonte com boa legibilidade. É importada do Google Fonts sob a licença Apache 2.0, é open-source e fornecida para ser usada gratuitamente em projetos pessoais e comerciais, sem necessidade de atribuição ou pagamento.

## Estrutura do Projeto

index.html - Arquivo HTML principal contendo a estrutura da página de login.

style.css - Arquivo CSS externo com os estilos da página.

## Como Executar

Clone este repositório ou baixe os arquivos.

Abra o arquivo index.html em um navegador da web.
